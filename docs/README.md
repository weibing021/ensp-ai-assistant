## 文档总览

本目录包含 eNSP AI Assistant 的产品与技术文档，建议配合项目根目录下的 `README.md` 一起阅读。

- `INTRODUCTION.md`：产品简介与典型使用场景。  
- `FEATURES.md`：核心特性列表（自然语言解析、AI 命令生成、拓扑、诊断等）。  
- `ARCHITECTURE.md`：高层架构说明与 Mermaid 架构图。  
- `USER_GUIDE.md`：典型使用流程 / 用户操作指南。  
- `STRUCTURE.md`：项目目录结构与各目录职责说明。  
- `PRODUCT_OVERVIEW.md`：以上文档的索引入口，方便跳转。
